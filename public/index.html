O stronie

Strona tworzona z wykorzystaniem HTML5, CSS3 (SASS), czystego JavaScriptu. Dane pogodowe pobierane są z API OpenWeatherMap (w celu ukrycia klucza API stosowane są Lambda Functions w Netlify). Do wyszukiwania położenia miast wykorzystywane jest Google Maps JavaScript API.
